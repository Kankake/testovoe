Данное приложение представляет собой систему управления библиотекой. Вот основной функционал:

1.Добавление книг в библиотеку с указанием названия, автора и года издания.
2.Удаление книг из библиотеки по их уникальному идентификатору.
3.Поиск книг по названию, автору или году издания.
4.Изменение статуса книги (в наличии или выдана).
5.Просмотр списка всех книг в библиотеке.
6.Сохранение данных библиотеки в JSON-файл и загрузка из него.
7.Интерактивный пользовательский интерфейс для выполнения всех вышеперечисленных операций.
Приложение обрабатывает различные ошибки, такие как неверный ввод данных, попытка удаления несуществующей книги или изменения статуса на недопустимый.
Оно также автоматически присваивает уникальные идентификаторы новым книгам при их добавлении в библиотеку.
